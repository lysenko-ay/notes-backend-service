# Users microservice

## Commands
| Action | Command |
| ------ | ------- |
| execute | `npm run start` |
| run dev | `npm run dev` |
| build docker container | `docker build . -t <image-name>` |
| run docker container | `docker run -p <PORT>:8080 <image-name>` |

## Documentation
API documentation: `http://localhost:8080/api-docs`
